# DataBase
***

# <u> 1. Usersテーブル </u>

| column   | type    | option         |
|:---------|:--------|:---------------|
| id       | integer | null: false    |
| name     | string  | null: false    |
| email    | string  | null: false    |
| password | string  | null: false    |
| avator   | string  | null: false    |
| member   | string  | null: false    |
| profile  | text    | null: false    |
| work     | string  | null: false    |

### Association
- has_many :prototypes
- has_many :likes
- has_many :comments

***
